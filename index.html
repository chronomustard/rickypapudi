Featured Projects

IoT Aquarium Temperature Controller

A system that is capable in adjusting aquatic environment temperature. It is created with Wemos D1 Mini microcontroller and Blynk. This device is able to adjust temperature inside a 19L volume aquarium from 25°C to 31°C. It is also able to datalog and access its controls wirelessly using Blynk.

EmissonChecked: Industrial Emission Quality Classifier

A classifier for industrial emission in accordance with a policy by Kementrian Lingkungan Hidup dan Kehutanan (KLHK). It is a data structure-based program using binary tree sort and search algorithm. It is created with C programming language.
