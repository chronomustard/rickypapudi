IoT Aquarium Temperature Controller

Jun 2022 - Jul 2022

This project is an IoT system that is capable in adjusting aquatic environment temperature. It is created with Wemos D1 Mini microcontroller and Blynk. This device is able to adjust temperature inside a 19L volume aquarium from 25°C to 31°C. It is also able to datalog and access its controls wirelessly using Blynk.
